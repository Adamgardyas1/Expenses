# Nazwa Projektu
Interaktywna aplikacja webowa do rozliczania wydatków trzech osób.

## Pliki w projekcie
- `app.py` – główny kod aplikacji
- `requirements.txt` – lista bibliotek Python
- `data/` – folder z danymi
- `utils.py` – funkcje pomocnicze
